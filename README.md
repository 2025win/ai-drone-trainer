# 🤖 智仿星伴 - AI无人机精准作业虚拟仿真实训系统

## 🎯 项目简介

基于人工智能技术的农用无人机虚拟仿真实训平台，深度融合AI学伴指导与虚拟仿真操作，为职业教育提供创新的教学解决方案。

## ✨ 核心特色

- **🤖 AI智能学伴**：专业问题实时解答与个性化指导
- **🎮 虚拟仿真**：安全的无人机操作训练环境
- **📚 模块化学习**：四大实训模块覆盖完整作业流程
- **🔬 深度交互**：基于认知科学的教学交互设计

## 🚀 技术架构

- **前端**：纯HTML/CSS/JS，响应式设计
- **部署**：Vercel + GitHub自动化部署
- **交互**：实时AI对话 + 虚拟仿真操作

## 📦 部署说明

1. 将文件上传到GitHub仓库
2. 在Vercel中连接GitHub仓库
3. 自动部署，获得生产环境网址

## 🎮 使用指南

1. **选择实训模块**：点击左侧四大模块开始学习
2. **AI对话学习**：在右侧与AI学伴实时交互
3. **虚拟仿真操作**：在底部进行无人机操作练习
4. **深度知识探索**：通过提问深入理解原理
